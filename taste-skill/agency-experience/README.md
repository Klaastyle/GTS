# Nexus: Premium Digital Experience

He diseñado y construido una experiencia digital de clase mundial que trasciende lo convencional. Siguiendo las directrices de **Taste Skill**, este proyecto utiliza un ecosistema tecnológico de vanguardia: **React 19**, **Tailwind CSS v4**, **GSAP** y **Phosphor Icons**.

## 🏗️ Arquitectura de Diseño (Taste Skill)

Para lograr un acabado premium, he aplicado los siguientes principios:
- **Asimetría Controlada (DESIGN_VARIANCE: 8):** El Hero section evita la simetría central común, optando por una composición dividida que guía la mirada de forma dinámica.
- **Cinemática GSAP (MOTION_INTENSITY: 6):** Implementación de *ScrollTrigger* para orquestar desplazamientos horizontales y revelaciones fluidas.
- **Bento 2.0:** Los componentes de funcionalidades utilizan una cuadrícula asimétrica con micro-interacciones perpetuas para mantener la interfaz "viva".
- **Liquid Glass:** Efectos de refracción realistas con desenfoque de fondo, bordes internos sutiles y sombras de difusión.

## 🚀 Stack Tecnológico

- **React 19 & Vite:** La base para una renderización ultra-rápida.
- **Tailwind CSS v4:** Utilizando el nuevo motor de diseño basado en variables CSS y el plugin de Vite.
- **GSAP:** Para la coreografía de scroll y animaciones de entrada.
- **Framer Motion:** Para micro-física de muelles (spring physics) en componentes UI.
- **Typography:** Uso estricto de *Outfit* para titulares y *JetBrains Mono* para datos técnicos.
